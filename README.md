# niubility-cloud

## 项目介绍
本项目基于spring alibaba cloud 微服务架构
## 系统架构图

## 组织结构

``` lua
niubility-cloud
├── niubility-commons -- 工具类及通用代码模块       
├── niubility-domain-models -- 子业务领域划分dubbo模块
├── niubility-gateway -- 基于Spring Cloud Gateway的微服务API网关服务
└── niubility-manage -- 基于spring alibaba cloud的微服务平台模块
```
